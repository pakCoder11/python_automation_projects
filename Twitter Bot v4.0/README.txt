============================
Twitter Auto-comment Bot
============================

REQUIREMENTS:
----------------
- Python 3.10+
- Google Chrome Updated Version
- Fast Internet connection 
- Microsoft Windows 10/11


HOW BOTS WORK
------------------
- Bot takes multiple twitter accounts, data load through account.csv 
- Bot select random comment at a time through comments.txt
- Bot select random hashtag at a time through hashtags.txt
- limit set by user, setting "comments_limit" variable in data.json file 

Bot takes account one by one and enter hashtags in latest tweet section, one by one bot automtically do auto-reply on comments and scrolldowns, this process continues until bot reachs the limit.
